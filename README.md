# Intelligent Resume Keyword Analyzer

A Java console-based ATS-style resume keyword analyzer that compares resume content with a job description and calculates a keyword match percentage.

## Features

- Add a job description
- Add resume content
- Compare resume keywords with job description keywords
- Calculate keyword match percentage
- Display matched keywords
- Save entered job and resume content to text files

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Java Collections Framework
- File Handling

## Java Concepts Used

- Abstract Classes
- Inheritance
- Interfaces
- Polymorphism
- HashMap
- HashSet
- ArrayList
- Exception Handling

## How to Run

### 1. Clone the repository

```bash
git clone https://github.com/shashwati-01/Intelligent-Resume-Keyword-Analyzer.git
```

### 2. Open the project directory

```bash
cd Intelligent-Resume-Keyword-Analyzer
```

### 3. Compile the Java program

```bash
javac ATSProject.java
```

### 4. Run the program

```bash
java ATSProject
```

## Program Menu

```text
===== ATS MENU =====
1. Add Job
2. Add Resume
3. Analyze Resume
4. Exit
```

Add a job description and resume using their respective IDs. Then select the Analyze Resume option and enter the Job ID and Resume ID.

The program compares the resume content with the job description, calculates the keyword match percentage, and displays the matched keywords.
